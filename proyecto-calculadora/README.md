# Calculadora PyQt6

Aplicación de calculadora básica con interfaz gráfica desarrolla en **Python** y **PyQt6**

## Características

*   Interfaz intuitiva y responsive.
*   Realiza operaciones básicas (suma, resta, multiplicación, división).
*   Función de borrar todo(AC), borrar(C), doble cero(00)

## Instalación y uso

1.  Asegurate de tener **Python 3.12** instalado.
2.  Clona este repositorio o descarga el código.
3.  Instala la librería necesaria:
    ```bash
    pip install PyQt6
    ```
4.  Ejecute la aplicación:
    ```bash
    python calculadora.py
    ```
